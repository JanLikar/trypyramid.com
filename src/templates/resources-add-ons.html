Pyramid add-ons

Pyramid add-ons extend the functionality of Pyramid, such as adding an ORM, sending email, or using a template-based language. The Python Package Index trove classifier "Framework :: Pyramid" is used by over 350 packages. Official add-ons and add-ons from the comunity are listed below.

Supported add-ons

The following Pyramid add-ons are officially endorsed by the Pylons Project. Documentation for each add-on is hosted at its respective name under the Pylons Project.

Community Pyramid add-ons and web-related packages

The following list of packages includes popular and useful Pyramid add-ons and web-related packages from the Pyramid and open source software communities, categorized to describe their primary function. Most packages depend on Pyramid, and others may work with multiple web frameworks including Pyramid.

These libraries are not supported by the Pylons Project, but have varying degrees of tests, documentation, support, and maintenance accorded by their maintainers. Consult the library's project page for details.
